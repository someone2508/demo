PLAYER ANALYTICS ASSIGNMENT - SUBMISSION

FILES INCLUDED:
1. player_analysis.py - Main analysis script
2. analysis_summary.txt - Summary of findings and recommendations
3. requirements.txt - Python dependencies
4. README.txt - This file

SETUP INSTRUCTIONS:

1. Install dependencies:
   pip install -r requirements.txt

2. Download the 5 CSV files from the Google Drive link:
   https://drive.google.com/drive/folders/1hgJPs06K4Q6wyFnYpBYvTVN8hYkJLW3e

3. Place the CSV files in a folder named 'data' in the same directory as player_analysis.py

4. Update the file names in player_analysis.py (lines 16-20) if needed to match your CSV file names

5. Run the analysis:
   python player_analysis.py

OUTPUT:
- Console output with all analysis results
- player_analysis_charts.png - Visualization of key metrics

The script will automatically:
- Load and process the data files
- Perform funnel & conversion analysis
- Analyze retention & engagement patterns
- Segment players by deposit value
- Generate visualizations

If data files are not found, the script will run with sample data for demonstration purposes.
